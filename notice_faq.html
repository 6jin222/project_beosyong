FAQ

총 35건의 게시물이 있습니다.
비밀번호를 잊어버렸어요.
버숑 홈페이지 - 로그인 - 회원 정보 찾기 - 회원 가입 시 등록하신 이메일 주소를 입력해주세요.

정상적으로 발송되었다면 입력하신 이메일의 메일함을 확인 바랍니다.
('받은 메일함'으로 전송된 내역이 없다면 '스팸 메일함'에서 확인 부탁드립니다.)

회원 아이디와 비밀번호를 확인 후 변경을 눌러 주시면 됩니다.

만약, 회원 가입 시 등록하신 이메일 주소도 잊어버렸다면 홈페이지 하단 'contact@growthchain.com'으로 문의 또는 카카오톡 채팅에서 '버숑'을 검색하여 버숑채널에서 문의주시면 확인 후 안내해 드리도록 하겠습니다.
제 아이디는 어디서 확인하나요?
[ PC 버전 ]
버숑 홈페이지 우측 상단 '정보수정' 클릭 후 '회원정보수정'에서 확인 가능합니다.

[ 모바일 버전 ]
버숑 홈페이지 왼쪽 상단 메뉴 클릭 후 '회원정보수정'에서 확인 가능합니다.

* 카카오톡 : kakao_ 로 시작하는 아이디
* 네이버 : naver_ 로 시작하는 아이디

아이디는 버숑의 모든 캠페인에서 사용됩니다.
